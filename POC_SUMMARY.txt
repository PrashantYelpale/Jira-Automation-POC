WHAT THE AUTOMATION DOES DAILY:

1. Completes 3-5 Tickets Per Developer
   - Randomly selects assigned tickets
   - Transitions them to Done status
   - Simulates daily work completion

2. Adds 5 New Tickets to Backlog
   - Uses task names from your Excel sheet
   - Random complexity and priority
   - 30% chance of priority override scenario
   - All unassigned initially

3. Assigns Unassigned Tickets
   - Loads developer readiness scores
   - Checks current workload in real-time
   - Calculates best match using formula:
     Score = (Readiness √ó 10) + Availability - Workload Penalty
   - Assigns to highest scoring developer
   - Adds explanation comment

4. Handles Priority Override
   - Detects when requester sets wrong priority
   - Adds comment explaining override
   - Assigns with correct priority
