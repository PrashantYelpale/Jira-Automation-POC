EASIEST WAY TO PUSH TO GITHUB

==============================================================================
OPTION 1: DOUBLE-CLICK THE BATCH FILE (EASIEST!)
==============================================================================

1. Find file: push_to_github.bat (in this folder)
2. Double-click it
3. When prompted, enter:
   Username: prashantyelpale63@gmail.com
   Password: [Your GitHub Personal Access Token]

Note: GitHub doesn't accept regular passwords anymore!
You MUST use Personal Access Token.

How to get token:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Select: repo (all checkboxes)
4. Click "Generate token"
5. COPY the token
6. Use this as password
